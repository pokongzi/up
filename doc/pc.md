# PC端功能规划

## 一、资料下载模块

### 1.1 功能概述
提供考公相关资料的下载服务，包括历年真题、备考资料、技巧总结等。

### 1.2 资料分类

#### （1）历年真题
- **按省份分类**：河南、上海、云南、内蒙古、北京、吉林、四川、天津、宁夏、安徽、山东、山西、广东、广西、新疆兵团、新疆区考、江苏、江西、河北、河南、浙江、海南、深圳、湖北、湖南、甘肃、福建、贵州、辽宁、重庆、陕西、青海、黑龙江
- **按科目分类**：申论、行测
- **按年份分类**：2025、2024、2023、2022...
- **按类型分类**：乡镇卷、县级卷、市级卷

#### （2）备考资料
- 行测母题集
- 技巧突破指南（判断推理、数资、言语、政治理论）

### 1.3 功能特性
- 资料搜索（按省份、年份、科目、类型关键词搜索）
- 资料预览（在线预览PDF前几页）
- 下载记录查询
- 批量下载（支持打包下载）

### 1.4 界面设计
- 左侧分类导航（省份/科目）
- 右侧资料列表（缩略图、名称、年份、类型、下载按钮）
- 顶部搜索栏
- 筛选条件（年份、类型）

## 二、真题在线查看模块

### 2.1 功能概述
在线浏览历年真题及答案，支持多机构答案对比。

### 2.2 题目展示
- **题目列表**
  - 省份/年份/地区三级筛选
  - 列表展示：年份、省份、类型、题目预览
  - 分页加载

- **题目详情**
  - 题目内容（文字/图片）
  - 材料内容
  - 问题题干
  - 字数要求

### 2.3 答案展示
- **多机构答案对比**
  - 华图答案
  - 粉笔答案
  - 中公答案
  - 答案高亮差异对比

- **答案解析**
  - 得分要点
  - 解析说明
  - 参考答案

### 2.4 功能特性
- 题目收藏（添加到收藏夹）
- 题目笔记（添加个人笔记）
- 答案纠错（用户反馈）
- 分享功能（生成链接/海报）

### 2.5 界面设计
- 顶部筛选栏（省份→年份→类型）
- 左侧题目列表
- 右侧题目详情+答案对比
- 底部导航（上一题/下一题）

## 三、在线模拟模块

### 3.1 功能概述
提供全真模拟考试环境，计时作答，智能评分。

### 3.2 模拟考试类型

#### （1）全真模拟
- 严格按考试时间
- 完整题目数量
- 倒计时提醒
- 自动交卷

#### （2）专项练习
- 按模块练习（归纳概括、对策建议、贯彻执行、大作文）
- 按难度练习
- 自由组合练习

#### （3）历年真题模拟
- 选择年份/省份真题进行模拟

### 3.3 考试流程
1. **选择试卷**
   - 试卷分类展示
   - 试卷预览（题目数量、建议用时）

2. **作答界面**
   - 顶部信息（考试名称、剩余时间）
   - 材料区域（可滚动）
   - 题目区域
   - 答题卡（显示已完成/未完成）
   - 标记功能（标记有疑问的题目）

3. **交卷**
   - 手动交卷
   - 到时自动交卷
   - 交卷确认（是否有未作答题目）

4. **成绩展示**
   - 总分
   - 各题得分
   - 各机构参考答案对比
   - 答案解析

### 3.4 智能评分（后期功能）
- 关键词匹配评分
- 语义相似度评分
- AI智能点评

### 3.5 答题记录
- 历史考试记录
- 成绩趋势图
- 错题本
- 答题数据分析

### 3.6 界面设计
- 首页：模拟考试入口、历年真题入口、我的记录
- 考试中：沉浸式作答界面
- 考试后：成绩展示、答案解析
